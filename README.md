# agent-stuff

Personal agent workflows, skills, and related tooling.

## Structure

- `skills/` — skill definitions and supporting references
- `extensions/` — Pi extension implementations

## Included skills

- `structured-dev` — a structured development workflow with multiple rigor levels

## Included extensions

- `structured-dev-workers` — supervised worker-per-task execution for approved `structured-dev` tasks, with child Pi JSON event mirroring and logs under `.pi/structured-dev/runs/`
